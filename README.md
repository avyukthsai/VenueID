# 🎤 VenueID – AI-Powered Venue Recommendation Web App

_HackAI Hackathon Project_

VenueID is a Python-based web application that leverages a Large Language Model (LLM) to recommend ideal venues and estimate audience sizes based on user-provided event descriptions, locations, and metrics. The system streamlines the event venue selection process through intelligent suggestions and an intuitive UI.

---

## 🚀 Features

✅ AI-powered venue recommendations  
✅ Audience size prediction from event details  
✅ Flask backend for LLM processing  
✅ Streamlit interface for quick testing and validation  
✅ React-based modern web UI for final user interaction  
✅ Built and pitched within the HackAI Hackathon timeframe

---

## 🛠️ Tech Stack

| Component                    | Technology Used                               |
| ---------------------------- | --------------------------------------------- |
| Backend                      | Python (Flask)                                |
| AI/LLM Integration           | Python-based LLM API (e.g., OpenAI or custom) |
| Prototype UI / Testing Layer | Streamlit                                     |
| Frontend                     | React                                         |
| Presentation                 | Hackathon pitch deck                          |

---

## 📍 How It Works

1. User inputs event location, description, and relevant metrics.
2. Flask backend processes the data and queries the LLM.
3. AI generates tailored venue suggestions and estimated audience size.
4. Results are first tested via Streamlit prototype.
5. Final UI displays results using a React-based frontend.
